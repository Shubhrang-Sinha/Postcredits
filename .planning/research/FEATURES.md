# Features Analysis

**Analysis Date:** 2026-04-24

## Table Stakes (Must Have)

### Identity Management

- User registration (email, password)
- User login with session
- Password hashing

### Media Catalog

- Books: title, author, pages, year, genres
- Movies: title, director, duration, year, genres
- Unified view of both media types

### Rating System

- User rates books/movies (1-5 scale)
- Timestamps for each rating
- One rating per user per work

### Spotistats Dashboard

- Most-watched genres
- Most-read genres
- Highest-rated years

## Differentiators (Competitive)

### Blend Recommendations

- Compare user profiles via SQL
- Find similar users
- Generate discovery list

### Taste Similarity Score

- Numeric similarity between users
- Based on shared ratings

## Anti-Features (Will NOT Build)

- Real-time chat
- OAuth login
- Mobile app
- Video streaming
